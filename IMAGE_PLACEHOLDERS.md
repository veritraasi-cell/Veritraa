## 📊 IMAGE SPECIFICATIONS

### Image Dimensions
```
Product Thumbnail (if needed): 400x400 px
Product Gallery Images: 800x800 px
Recommended Format: JPG or PNG
Recommended File Size: 200-500 KB per image
```

### Image Recommendations
- **Background**: Clean, white or light background
- **Quality**: High resolution (minimum 400x400)
- **Perspective**: Top-down view for powder/spices
- **Lighting**: Natural, bright lighting
- **Packaging**: Show actual packaging if available
- **Variants**: Include images of different pack sizes

---

## 🚀 HOW TO REPLACE IMAGES

### Step 1: Prepare Your Images
1. Take photos of your spices in good lighting
2. Resize to 400x400 px (main) and 800x800 px (gallery)
3. Save as `.jpg` format
4. Name according to the pattern: `{product-id}.jpg`, `{product-id}-2.jpg`, etc.

### Step 2: Add Images to Project
```
1. Create folder: public/images/products/
2. Copy all images into this folder
3. File names must match exactly (case-sensitive on Linux/Mac):
   - shahi-garam-masala.jpg
   - shahi-garam-masala-2.jpg
   - kanda-lasun-chutney.jpg
   - etc.
```

### Step 3: Test in Browser
```
1. Start dev server: npm run dev
2. Open: http://localhost:3000/shop
3. Verify all images load correctly
4. Check responsive design on mobile
```

### Step 4: Update Product Page (Optional)
If images are already referenced in code (lib/products.ts), they will automatically use the new images when files are added to the folder.

## 🔍 TROUBLESHOOTING

### Images Not Loading?
1. Check file path: `/images/products/filename.jpg`
2. Verify file names match exactly (case-sensitive)
3. Ensure `public/` folder exists
4. Restart dev server: `npm run dev`
5. Clear browser cache (Ctrl+Shift+Delete)

### Images Too Large?
1. Compress images (use TinyPNG or ImageOptim)
2. Keep below 500 KB per image
3. Use JPG format for photos
4. Use PNG for images with transparency

### File Path Issues
- **Windows**: Use forward slashes: `/images/products/...`
- **Mac/Linux**: Use forward slashes: `/images/products/...`
- **Never use backslashes**: `\images\products\...` ❌

## 🎨 IMAGE NAMING CONVENTION

All product images follow this pattern:

```
{product-id}.jpg              - Main product image
{product-id}-2.jpg            - Secondary angle/detail
{product-id}-3.jpg            - Third variation (if needed)
{product-id}-4.jpg            - Fourth variation (if needed)
```

**Examples:**
- ✅ `shahi-garam-masala.jpg`
- ✅ `shahi-garam-masala-2.jpg`
- ❌ `Shahi garam masala.jpg` (capital letters)
- ❌ `shahi_garam_masala.jpg` (underscores, use hyphens)
- ❌ `shayi-garam-masala.jpg` (typo)
